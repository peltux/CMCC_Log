# CMCC 自动上\下线脚本

## 1. 安装依赖

```shell
pip install requests plyer
```

将两个文件中的 `USERNAME` 和 `PASSWORD` 改为自己的账号密码。

## 2. 运行

- 上线：

  ```bat
  python login.txt
  ```

- 下线：

  ```bat
  python logout.txt
  ```

## 3. 登录时自启动

1. **创建登录脚本**（如果是 `login.py`，确保它在固定路径，如 `C:\scripts\login.py`）。
2. **创建任务**
   - 按 `Win + R`，输入 `taskschd.msc` 回车，打开任务计划程序。
   - 右侧点击 **创建任务**，名称填 `校园网自动登录`，勾选 **不管用户是否登录都要运行**（如需要后台静默），配置中选择 **Windows 10/11**。
3. **设置触发器**
   - 切换到 **触发器** 选项卡 → **新建**。
   - 开始任务选择 **启动时**，或更稳妥的 **在工作站解锁时**。
   - 勾选 **延迟任务时间**，填入 `2 秒`（给网卡获取 IP 留足时间）。
4. **设置操作**
   - 切换到 **操作** 选项卡 → **新建**。
   - 操作选择 **启动程序**。
   - 程序或脚本：`python`（或完整路径如 `C:\Users\你的用户名\AppData\Local\Programs\Python\Python39\python.exe`）。
   - 添加参数：`"C:\scripts\login.py"`（你的脚本路径）。
5. **条件与设置**
   - **条件**卡片：取消勾选 **只有在计算机使用交流电源时才启动此任务**（避免笔记本电池模式不触发）。
   - **设置**卡片：勾选 **如果任务失败，按以下频率重新启动**，可设为 `1 分钟`，最多重试 `3 次`。
6. 点击确定，输入你的 Windows 登录密码，任务创建完毕。

重启电脑测试，等待 30 秒后应会收到登录通知。
